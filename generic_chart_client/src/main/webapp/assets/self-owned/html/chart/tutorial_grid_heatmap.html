笛卡尔坐标系上的热力图

目的

用来展示基于笛卡尔坐标系的数据热度

数据原型

  • X坐标轴:字符串类型的数组,比如['12a', '1a', '2a', '3a', '4a', '5a', '6a','7a', '8a', '9a','10a','11a','12p', '1p', '2p', '3p', '4p', '5p','6p', '7p', '8p', '9p', '10p', '11p']
  • Y坐标轴:字符串类型的数组,比如['Saturday', 'Friday', 'Thursday','Wednesday', 'Tuesday', 'Monday', 'Sunday']
  • 热度最小值:数字，默认为0
  • 热度最大值:数字，默认为10
  • 基于Cell的热度值：基于[X，Y，Value]组成二维数组，比如[[0,0,5],[0,1,1],[0,2,0],[0,3,0]]

如何编辑此种类型的热力图

  • 设置X坐标轴:输入相应的设置值，用逗号分割，系统会转换为对应的数据原型数值,比如x,y,z
  • 设置Y坐标轴:输入相应的设置值，用逗号分割，系统会转换为对应的数据原型数值,比如a,b,c
  • 设置热度范围:输入对应的最大最小值
  • 批量设置热度值:输入基于[X坐标index，Y坐标index，Value]组成二维数组,比如[[0,0,5],[0,1,1],[0,2,0],[0,3,0]]
  • 增加热度值:输入X坐标value,Y坐标value,热度值
  • 移除热度值:输入X坐标value,Y坐标value

注意

  • 批量设置热度值时，输入X，Y坐标轴的index
  • 单次设置热度值时，输入X，Y坐标轴的value

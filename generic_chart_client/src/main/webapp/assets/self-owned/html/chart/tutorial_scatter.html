散点（气泡）图

直角坐标系上的散点图可以用来展现数据的 x，y 之间的关系，如果数据项有多个维度，其它维度的值可以通过不同大小的 symbol 展现成气泡图，也可以用颜色来表现。

数据模型样例

  • Axis: 2013, 2014, 2015, 2016, 2017
  • Data: [{"name":"GDP","data":[20, 30, 30, 50, 80]},{"name":"KPI","data":[60, 90, 10, 80, 70]},{"name":"AI","data": [10, 20, 30, 40, 50]},{"name":"BI","data": [10, 20, 30, 40, 50]}]
